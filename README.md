# Responsive Product Catalog with Dynamic URL Filtering

A responsive, high-performance React application built with Vite featuring a dynamic product listing grid, sidebar toggle, custom pagination, and multi-criteria client-side filtering (by category, brand, and price range).

This project implements robust state persistence across page routing and browser history navigation by utilizing the browser's URL search parameters as the single source of truth.
